Это приложение на сервлетах, позволяющее зарегистрировать пользователя и работать с таблицей employees. 
(пароль: не менее 6 символов, логин: не менее 4 символов, уникальный)
В базе данных 2 таблицы:
profile (хранит данные о зарегистрированных пользователях: id, login, password_hash)
employees (таблица с который может работать пользователь. Реализованы CRUD операции. 
id_profile соответствует id текущего пользователя. Каждый пользователь видит только свои записи в таблице employees)
