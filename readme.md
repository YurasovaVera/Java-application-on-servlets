Это приложение на сервлетах, позволяющее зарегистрировать пользователя и работать с таблицей employees. 
(пароль: не менее 6 символов, логин: не менее 4 символов, уникальный)

**В базе данных 2 таблицы:**
1. profile (хранит данные о зарегистрированных пользователях: id, login, password_hash)
2. employees (таблица с который может работать пользователь. Реализованы CRUD операции. id_profile соответствует id текущего пользователя. Каждый пользователь видит только свои записи в таблице employees)

**После запуска** 
docker-compose up --build 
**необходимо ввести в url**
http://localhost:8080/myapp/login

Откроется страница входа, после нажатия на кнопку "Create account" можно будет зарегистрироваться, а затем войти в аккаунт и приступить к работе с таблицей.
